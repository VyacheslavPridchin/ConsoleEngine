# ConsoleEngine

The ConsoleEngine library extends the graphical capabilities of the standard console. ConsoleEngine.UI contains
interactive graphic elements: Label, Button, Border. It also reports mouse and console status in the console.

***

# ConsoleEngine

Библиотека ConsoleEngine расширяет графические возможности стандартной консоли. ConsoleEngine.UI содержит в себе
интерактивные графические элементы: Label, Button, Border. Также сообщается о состоянии мыши и консоли в консоли.
